# monitor
可拖拽大屏演示系统，目的是为了能够方便大屏系统的快捷生成，通过拖拽的方式完成图表的搭建。
不用关心设置图表的尺寸，拉伸图表即可控制大小。
能够保存图表在页面上的位置信息，大小信息，下次打开时可还原上次图表状态。
能够通过文字提示的方式控制图表显示的参数信息，而不需要去代码配置它。
支持全屏预览效果。
支持配置文件信息的下载。
支持主题添加，内置了 `e-charts` 的一些主题，选择后可以直接在图表上应用。
保存文件
编辑文件
删除文件

## 基础图表
- 基础图表选择：
  - 折线图
  - 柱状图
  - 饼图
- 配置项：
  - 标题组件
  - 图例组件

- 自定义背景上传
